# tft-arduino
Auto-generative music and dynamic patterns with Arduino.

## Description
Minimalistic USB MIDI gadget developed with Arduino UNO, TFT display and rotary encoder.

## Wiring

Before doing any wiring please check the voltages. 

### TFT

VCC - 5v  
GND - GND  
CS - D10  
DC/RST - D9  
A0 - D8  
SDA -D11  
SCK - D13  
LED - 3.3v  

### Encoder

BTN - D5  
CW - D3  
CWW - D2
